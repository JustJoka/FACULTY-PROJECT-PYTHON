🍔 Sistema de Controle de Pedidos para Lanchonete<br>
<br>
Aplicação desenvolvida em Python para gerenciamento de pedidos em lanchonetes. O sistema permite registrar, visualizar e organizar pedidos através de uma interface simples, utilizando banco de dados para armazenamento das informações.<br>
<br>
O projeto foi desenvolvido com foco em aprendizado prático de programação em Python, interfaces gráficas e integração com banco de dados.<br>
<br>
🚀 Funcionalidades<br>
<br>
Cadastro de pedidos<br>
<br>
Visualização de pedidos registrados<br>
<br>
Gerenciamento de informações de clientes<br>
<br>
Armazenamento de dados em banco de dados<br>
<br>
Organização dos dados para controle de pedidos<br>
<br>
🛠️ Tecnologias Utilizadas<br>
<br>
Python

Tkinter

SQLite
